# Installation

DUBSAR is not publicly installable from this repository yet.

The private beta package requires a coordinated DUBSAR Desktop/local runtime and Claude Code plugin. The Marketplace package is still under review and has not been published.

---

## Planned beta flow

The intended tester journey is:

1. Receive controlled beta access.
2. Install DUBSAR Desktop and its local runtime.
3. Add this repository as the DUBSAR Claude Code Marketplace.
4. Install the DUBSAR plugin.
5. Approve the required Claude Code plugin and MCP prompts.
6. Start DUBSAR from Claude Code.
7. Activate or verify beta access through the approved local flow.
8. Resume an existing Mission or create a new one.

The final commands will be published only after the package and repository name are validated.

---

## Current prerequisites under review

- supported Claude Code version;
- supported operating systems;
- DUBSAR Desktop/runtime package;
- access or activation state;
- network access to the private service where required;
- clean uninstall and upgrade behavior.

---

## Internal compatibility names

Early beta diagnostics may display internal `scribe` identifiers such as `scribe-mcp` or `/scribe-*` commands.

These are compatibility identifiers, not a second product name. Do not manually rename executables, local paths or command identifiers unless an official migration is published.

---

## Do not use legacy commands

Installation commands from earlier SCRIBE staging material are not canonical DUBSAR instructions.

Do not install from an unpublished legacy Marketplace path or copy a private staging package into a public repository manually.
