# Public RFCs

This folder contains public technical notes for SCRIBE Builder.

They are conceptual and non-sensitive.

They describe the public shape of the protocol without exposing the private engine, internal gates, prompts, sealed journals, proof artifacts or implementation details.

## Status

These RFCs are not product documentation.

They are public design references for a project that remains experimental and not commercial-ready.

## Boundary

The RFCs intentionally avoid:

- private engine code;
- internal validation policies;
- signing keys or trust material;
- real audit logs;
- sealed journals;
- private prompts;
- provider details;
- operational write mechanisms;
- sensitive implementation choices.

They define public concepts only.
